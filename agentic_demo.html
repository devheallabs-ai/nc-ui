How NC Agents Work

The Plan-Act-Observe loop — built into the language

1. Plan

Agent analyzes the task and decides which tools to use. NC AI provides intent-aware reasoning.

2. Act

Agent calls exactly ONE tool per step with structured JSON: {"action": "search", "input": {...}}

3. Observe

Agent reads tool output, decides to continue or finish. Uses {"action": "finish", "result": "..."} when done.
