# Changelog — NC Language Tools (VSCode Extension)

All notable changes to the NC Language Tools VSCode extension will be documented here.

## [0.2.0] — 2026-03-30

### Added
- Syntax highlighting for `.nc` and `.ncui` files
- Code runner with sidebar console output
- LSP integration for diagnostics and completion
- Snippet library for NC patterns

### Fixed
- Improved tokenization for multi-line strings
- Better error highlighting in `.ncui` files
